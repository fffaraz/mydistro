konversation-25.12.2

Introduction to konversation

The konversation package is a KDE Frameworks-based IRC client.

Note

This package is known to build and work properly using an LFS 13.0 platform.

Package Information

  • Download (HTTP): https://download.kde.org/stable/release-service/25.12.2/src/konversation-25.12.2.tar.xz

  • Download MD5 sum: b9c6f8769c1b9442a774fd451e3177b0

  • Download size: 4.3 MB

  • Estimated disk space required: 72 MB

  • Estimated build time: 0.6 SBU (using parallelism=4)

konversation Dependencies

Required

KDE Frameworks-6.23.0

Installation of konversation

Install konversation by running the following commands: 

mkdir build &&
cd    build &&

cmake -D CMAKE_INSTALL_PREFIX=$KF6_PREFIX \
      -D CMAKE_BUILD_TYPE=Release         \
      -D BUILD_TESTING=OFF                \
      -W no-dev .. &&
make

This package does not come with a test suite.

Now, as the root user: 

make install
